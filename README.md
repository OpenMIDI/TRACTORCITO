# TRACTORCITO
Mezclador para TRAKTOR DJ hecho con Arduino y el Kilomux shield

Este controlador MIDI fue fabricado por Yaeltex (http://yaeltex.com.ar) como primer prototipo para el TCMIDI (Taller de controladores MIDI).
Es un controlador compacto y simple que reune los controles generales de TRAKTOR. Utiliza el Kilomux shield para la conexi√≥n de los componentes a la Arduino y tiene desarrollado un MIDI Mapping especial para el software.
